# 📄 Мастер документов — Telegram бот

Бот для автоматической генерации PDF-документов: счетов на оплату и актов выполненных работ.

## 🚀 Возможности

- **Создание счетов** — генерация профессиональных счетов на оплату в формате PDF
- **Создание актов** — акты выполненных работ с подписями сторон
- **История документов** — все созданные документы сохраняются в базе данных
- **Отправка администратору** — копия каждого документа отправляется администратору
- **Простой интерфейс** — пошаговый диалог для создания документа

## 📋 Требования

- Python 3.9+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

## ⚙️ Установка

### 1. Клонирование или копирование проекта

```bash
cd 03-bot-docs
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

### 3. Активация виртуального окружения

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 5. Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```bash
copy .env.example .env
```

Отредактируйте `.env` и укажите:

```env
# Токен бота (получить у @BotFather)
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

# ID администратора (узнать у @userinfobot)
ADMIN_ID=123456789
```

### 6. Запуск бота

```bash
python bot.py
```

## 📖 Использование

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запустить бота, показать главное меню |
| `/help` | Показать справку по боту |
| `/mydocs` | Показать список моих документов |
| `/cancel` | Отменить текущее действие |

### Кнопки главного меню

- **🧾 Создать счёт** — начать процесс создания счёта на оплату
- **📄 Создать акт** — начать процесс создания акта выполненных работ
- **📁 Мои документы** — просмотреть историю созданных документов
- **📞 Помощь** — показать справку

### Процесс создания документа

1. Выберите тип документа (счёт или акт)
2. Введите номер документа
3. Введите дату (или нажмите Enter для сегодняшней)
4. Введите описание товара/услуги
5. Введите сумму
6. Введите плательщика/заказчика
7. Проверьте данные и подтвердите

После подтверждения бот отправит готовый PDF-файл.

## 🔧 Настройка

### Получение токена бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен в `.env`

### Получение Admin ID

1. Откройте [@userinfobot](https://t.me/userinfobot)
2. Отправьте любое сообщение
3. Скопируйте ваш ID в `.env`

## 📝 Примечания

- Все документы сохраняются в SQLite базе данных `documents.db`
- PDF файлы временно хранятся в папке `temp/`
- Копии всех документов отправляются администратору
